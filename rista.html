Rista

UI/UX Design & Graphic Design

Overview

Rista is a membership-based company targeting coffee lovers. With a Rista subscription, the user can get a cup of barista coffee every day, save money, support local shops, and discover new cafes at the same time.

My role: I was the solo designer, collaborating directly with the owners of the company.

Project goal: Redesign different screens on the mobile app and design from scratch a dashboard.

Tools: Adobe XD, Illustrator, Photoshop, Paper & Pencil

The Problem

The mobile app currently has some outdated pages that require refinement, along with several missing pages that need to be created. Additionally, the app will receive new features.

Some users have expressed confusion about certain features in the app, indicating a need for greater clarity in the design.

The Solution

The mobile app requires a makeover to bring it up to speed. The existing pages need a facelift to improve the visual appeal and navigation. In addition, new pages need to be added to provide a more comprehensive experience. The new features should be designed with the user in mind to ensure they are intuitive and easy to use. Consistency across all pages will provide a seamless user experience..

Challenges

• Communicating complex features in a clear and concise way.

• Balancing simplicity with functionality.

• Ensuring that all users, regardless of technical ability, can easily use the app.

• Creating designs that are visually appealing, but not overwhelming.

• Understand the user pain point.
